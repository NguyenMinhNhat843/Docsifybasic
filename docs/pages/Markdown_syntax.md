# Bảng cú pháp trong Markdown

| Cú pháp   | Ý nghĩa                     |
| --------- | --------------------------- |
| # (1, 6)  | Thẻ H1 tới H6               |
| >         | Tạo đoạn nội dung nhấn mạnh |
| \*, -     | Tạo list                    |
| 1. 2. 3.  | Tạo list số                 |
| [Name](#) | Tạo thẻ a                   |
| **Text**  | In đậm                      |
| `text`    | hightline                   |
| _text_    | In nghiêng                  |
